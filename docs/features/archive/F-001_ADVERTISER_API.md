# F-001: Advertiser API

**Status:** Complete
**Completed:** 2025-03-05
**Impact:** REST API to manage advertisers (job posters) via CRUD operations at `{host}/api/v1/advertisers/`.

## Problem

We need an API to create, read, update, and delete advertiser records. Advertisers represent job posters in the system. The API must be versioned and follow REST conventions.

## Solution

### Overview

- Create Django app `dj_advertisers` with an `Advertiser` model (basic fields only).
- Expose CRUD via Django REST Framework (DRF) ViewSets.
- Mount under `{host}/api/v1/`.
- No authentication or permission schema for this iteration.

### Advertiser Model (Basic Fields)

| Field | Type | Nullable | Default | Description |
|-------|------|----------|---------|-------------|
| `id` | `int` (PK) | NO | auto | Primary key |
| `name` | `VARCHAR(255)` | NO | — | Advertiser name |
| `discount` | `Decimal(5,4)` | NO | `0` | Discount rate |
| `default_conversion_rate` | `Decimal(6,4)` | NO | `0.01` | Default conversion rate |
| `monetization_type` | `ENUM(monetizationtype)` | NO | `'CPC'` | Monetization type |
| `billing_source` | `ENUM(billingsource)` | NO | `'INTERNAL'` | Billing source |
| `billing_timezone` | `ENUM(timezone)` | NO | `'UTC'` | Billing timezone |
| `created_at` | `DateTimeField` | NO | `timezone.now` | Creation timestamp |
| `updated_at` | `DateTimeField` | NO | `timezone.now` | Last update timestamp |

**Note:** BaseEntity audit fields (`created_by_actor_id`, `updated_by_actor_id`, `version`, `status`, etc.) are deferred to a later feature.

**PostgreSQL coupling:** Choice fields use native PostgreSQL ENUM types (`monetizationtype`, `billingsource`, `timezone`) instead of `VARCHAR` with Django `TextChoices`, improving type safety at the database level.

### API Design

- **Base URL:** `{host}/api/v1/`
- **Resource:** `advertisers`
- **Endpoints:**
  - `GET /api/v1/advertisers/` — List
  - `POST /api/v1/advertisers/` — Create
  - `GET /api/v1/advertisers/{id}/` — Retrieve
  - `PUT /api/v1/advertisers/{id}/` — Full update
  - `PATCH /api/v1/advertisers/{id}/` — Partial update
  - `DELETE /api/v1/advertisers/{id}/` — Delete

**PostgreSQL ENUM types (add to migration before CreateModel):**

Run `makemigrations` to generate the initial migration, then edit it to insert `RunSQL` as the first operation (enum types must exist before the table is created):

```python
# dj_advertisers/migrations/0001_initial.py — insert at start of operations
CREATE_ENUMS = """
CREATE TYPE monetizationtype AS ENUM ('CPC', 'CPA');
CREATE TYPE billingsource AS ENUM ('INTERNAL', 'PARTNER');
CREATE TYPE timezone AS ENUM ('UTC', 'EST', 'CST', 'PST');
"""

DROP_ENUMS = """
DROP TYPE IF EXISTS monetizationtype;
DROP TYPE IF EXISTS billingsource;
DROP TYPE IF EXISTS timezone;
"""

operations = [
    migrations.RunSQL(CREATE_ENUMS, DROP_ENUMS),
    migrations.CreateModel(...),  # auto-generated by makemigrations
]
```

**Custom field for PostgreSQL ENUM:**

```python
# dj_advertisers/fields.py
from django.db import models

class PostgreSQLEnumField(models.CharField):
    """CharField that maps to a native PostgreSQL ENUM type."""

    def __init__(self, enum_name, **kwargs):
        self.enum_name = enum_name
        super().__init__(**kwargs)

    def db_type(self, connection):
        if connection.vendor == "postgresql":
            return self.enum_name
        return super().db_type(connection)
```

**Model example (PostgreSQL-coupled):**

```python
# dj_advertisers/models.py
from django.db import models

from .fields import PostgreSQLEnumField

class MonetizationType(models.TextChoices):
    CPC = "CPC", "CPC"
    CPA = "CPA", "CPA"

class BillingSource(models.TextChoices):
    INTERNAL = "INTERNAL", "Internal"
    PARTNER = "PARTNER", "Partner"

class BillingTimezone(models.TextChoices):
    UTC = "UTC", "UTC"
    EST = "EST", "Eastern"
    CST = "CST", "Central"
    PST = "PST", "Pacific"

class Advertiser(models.Model):
    name = models.CharField(max_length=255)
    discount = models.DecimalField(max_digits=5, decimal_places=4, default=0)
    default_conversion_rate = models.DecimalField(max_digits=6, decimal_places=4, default=0.01)
    monetization_type = PostgreSQLEnumField(
        "monetizationtype",
        max_length=20,
        choices=MonetizationType.choices,
        default=MonetizationType.CPC,
    )
    billing_source = PostgreSQLEnumField(
        "billingsource",
        max_length=20,
        choices=BillingSource.choices,
        default=BillingSource.INTERNAL,
    )
    billing_timezone = PostgreSQLEnumField(
        "timezone",
        max_length=10,
        choices=BillingTimezone.choices,
        default=BillingTimezone.UTC,
    )
    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)
```

**Note:** `TextChoices` classes remain for Python-level validation and DRF serialization; `PostgreSQLEnumField` uses them for `choices` while mapping to native PostgreSQL ENUM columns.

## Unit Test Examples

**Model tests:**

```python
# dj_advertisers/tests/test_models.py
from django.test import TestCase
from dj_advertisers.models import Advertiser, MonetizationType, BillingSource, BillingTimezone

class AdvertiserModelTest(TestCase):
    def test_create_with_defaults(self):
        a = Advertiser.objects.create(name="Acme")
        self.assertEqual(a.discount, 0)
        self.assertEqual(a.monetization_type, MonetizationType.CPC)
        self.assertEqual(a.billing_source, BillingSource.INTERNAL)
        self.assertEqual(a.billing_timezone, BillingTimezone.UTC)
```

**API tests:**

```python
# dj_advertisers/tests/test_views.py
from rest_framework.test import APITestCase
from rest_framework import status
from dj_advertisers.models import Advertiser

class AdvertiserViewSetTest(APITestCase):
    def test_list_empty(self):
        r = self.client.get("/api/v1/advertisers/")
        self.assertEqual(r.status_code, status.HTTP_200_OK)
        self.assertEqual(r.json(), [])

    def test_create_and_retrieve(self):
        r = self.client.post("/api/v1/advertisers/", {"name": "Acme"}, format="json")
        self.assertEqual(r.status_code, status.HTTP_201_CREATED)
        pk = r.json()["id"]
        r2 = self.client.get(f"/api/v1/advertisers/{pk}/")
        self.assertEqual(r2.status_code, status.HTTP_200_OK)
        self.assertEqual(r2.json()["name"], "Acme")
```
